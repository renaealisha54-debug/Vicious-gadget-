# **App Name**: Vicious Archi

## Core Features:

- Intelligent Code Scanner: Automatically detect framework signatures and language types from any raw code input.
- Live Project Dissector: Parse monolith code blobs into logical file structures and sub-directories in real-time.
- AI Documentation tool: A tool that utilizes generative AI to reason through code and synthesize context-aware README files.
- Framework Scaffolding: Inject necessary boilerplate like package.json and tsconfig.json based on detected dependencies.
- Dependency Automator: Identify required libraries from code and generate scripts to install necessary dependencies.
- Interactive Tree Navigator: Explore the generated project architecture through a visual file-explorer interface.
- Instant Zip Export: Compile the entire processed file set into a ready-to-deploy ZIP archive for local use.

## Style Guidelines:

- Primary color: Tech Blue (#5190F2) for engineering precision.
- Background: Deep Charcoal (#14171C) for high-contrast focus.
- Accent: Electric Cyan (#17BDD9) for action guidance.
- Headlines: 'Space Grotesk'; Body: 'Inter'; Code: 'Source Code Pro'.
- Monolinear, geometric icons with a 1.5px stroke weight for an architectural feel.
- Split-pane workbench with input editor on left and hierarchical navigator on right.
- Subtle horizontal shifts and line-fades during file dissection.
