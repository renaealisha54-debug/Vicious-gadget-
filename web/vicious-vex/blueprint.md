# **App Name**: VexCode

## Core Features:

- Code Editor with Syntax Highlighting: An interactive web-based code editor featuring syntax highlighting for various programming languages to enhance readability and development experience.
- Project & File Management: Enables users to create, open, save, delete, and organize project files and directories directly within the web application.
- AI-Powered Code Suggestions: Provides intelligent code completions, error explanations, and code generation suggestions using a generative AI tool to assist in the development process.
- Integrated Build & Output Console: A console panel that connects to a backend service to execute compilation/build commands (e.g., for Java/Gradle projects) and displays real-time output and logs.
- Customizable Workspaces: Allows users to arrange, resize, and hide different development panels (e.g., file explorer, console, secondary editors) to personalize their coding environment.

## Style Guidelines:

- A dark theme for reduced eye strain during coding. Primary elements use a 'digital blue' hue (HSL H:240, S:70%, L:60%, converted to hex #6666E6). The background is a subtly darker variant of the same hue (HSL H:240, S:15%, L:12%, converted to hex #191923). An accent of bright cyan (HSL H:210, S:80%, L:70%, converted to hex #99E5FF) is used for highlights and interactive elements.
- For UI text and headlines, the 'Inter' sans-serif font is recommended for its modern, clean, and objective look. For all code display areas, 'Source Code Pro' monospace font will be used to ensure clear readability of programming constructs.
- Utilize simple, outline-style vector icons that convey clarity and professionalism, suitable for a development environment. Prioritize readability over heavy styling.
- A multi-panel layout with draggable and resizable sections for editor, file explorer, and console. Default arrangement should prioritize the code editor, with tool panels dockable or floatable, reminiscent of modern IDEs.
- Subtle and functional animations for UI interactions like panel resizing, tab switching, and console output updates, to provide smooth feedback without distraction.
